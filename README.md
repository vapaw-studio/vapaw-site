# VAPAW Homepage v1.0 Fullscreen

A static brand homepage designed for GitHub Pages / Cloudflare Pages.

## Page Structure

1. Hero — fullscreen video carousel
2. Universe — fullscreen brand image section
3. Product — fullscreen 3-product catalog section
4. Journal — fullscreen magazine-style content section
5. Footer

## Files

- `index.html`
- `style.css`
- `script.js`
- `assets/`

## Notes

- Keep product images as transparent PNG if possible.
- Recommended hero video: MP4 H.264, 8–15 seconds, muted, 1920×1080 or higher.
- Upload all files to GitHub, then deploy with GitHub Pages or Cloudflare Pages.
